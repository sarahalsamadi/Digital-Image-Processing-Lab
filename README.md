# 📚 Interactive Digital Image Processing Learning Series

An advanced educational dashboard built with **Streamlit** to demonstrate core concepts of Digital Image Processing (DIP) using Python. This project serves as a visual and interactive guide for students and developers to understand how images are manipulated at the pixel level.

## 🌟 Features
- **Multi-Module Learning:** Divided into 9 comprehensive lectures covering the entire DIP pipeline.
- **Real-time Processing:** Users can upload their own images and apply filters/transformations instantly.
- **RTL Support:** Designed with a custom CSS injection to support Arabic language and Right-to-Left (RTL) layouts perfectly.
- **Comprehensive Toolbox:** Includes everything from basic color spaces to advanced morphological operations and geometric transformations.

## 🎓 Curriculum Covered
1. **Introduction:** Digital image structure and bit-depth.
2. **Color Spaces:** Grayscale, HSV, and RGB manipulations.
3. **Point Operations:** Brightness, contrast, and histogram equalization.
4. **Filters & Kernels:** Blurring, sharpening, and noise reduction.
5. **Edge Detection:** Sobel, Canny, and Prewitt operators.
6. **Morphological Operations:** Dilation, Erosion, Opening, and Closing.
7. **Geometric Transformations:** Scaling, Rotation, and Affine transforms.
8. **Final Pipeline:** Building a complete end-to-end image enhancement workflow.

## 🛠️ Tech Stack
- **Frontend Framework:** Streamlit
- **Image Processing:** OpenCV, Pillow, NumPy
- **Visualization:** Matplotlib
- **Language:** Python 3.10+

## 📁 Project Structure
- `main.py`: The entry point and UI controller for the Streamlit app.
- `modules/`: Contains separate Python scripts for each lecture (e.g., `lecture1_intro.py`, `lecture2_colorspaces.py`).
- `assets/`: Stores default images and visual aids.
